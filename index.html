Unify your data processing

Maki Nage exposes a unified set of Reactive, Functional, and Descriptive APIs for:

  • Batch Data
  • Stream Data

Maki Nage processes all data in a reactive way, but considering a batch as a bounded stream.

One code for your whole workflow

Maki Nage can be used on a simple laptop, up to deployment.

  • Start exploring on your laptop
  • Scale localy on a single server or small cluster
  • Deploy your code on a production Kafka cluster
